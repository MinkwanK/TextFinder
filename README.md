# TextFinder
단어 찾기 (QT 예제 - 확장버전)

![image](https://github.com/user-attachments/assets/4f2513b4-26ca-4912-b256-1e69c94da60e)

Find: 현재 커서 다음의 단어를 찾기
Back: 현재 커서 앞의 단어를 찾기    (확장)
All: 텍스트 내부에서 모든 단어찾기  (확장)
CaseSensitive: 대소문자 구분       (확장)
